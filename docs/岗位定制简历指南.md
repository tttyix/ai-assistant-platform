**核心卖点**: 向量检索 + 混合检索 + 知识图谱 + GraphRAG

### AI 助手集成平台（调到第一）

```markdown
### AI 助手集成平台                                    2026.01-2026.03
项目角色：全栈开发 + RAG 系统开发
技术栈：FastAPI, LangChain, Qdrant, PostgreSQL, Redis

项目描述：
企业级 AI 助手平台，核心是基于 Qdrant 的 RAG 检索增强系统。

核心工作：
• 基于 LangChain 构建 RAG 检索增强系统：
  - 文档加载：支持 PDF/Word/Markdown/TXT 四格式
  - 文本分块：智能分块策略，500 字/块，重叠 50 字
  - 向量化：sentence-transformers 嵌入模型，384 维
  - 存储：Qdrant 向量数据库，HNSW 索引
• 实现混合检索：向量相似度 (60%) + 关键词 BM25(40%)
• 设计 Rerank 重排序机制，对 Top-20 结果重新打分
• 优化检索性能：响应时间 < 200ms，召回率 92%
• 基于 Qdrant 的 payload 过滤，支持知识库隔离

技术成果：
• 混合检索召回率 92%，Top-5 准确率 87%
• 语义检索响应 < 200ms，支持 4 种文档格式
• Rerank 重排序使准确率提升 15%
```

**调整点**:
- 🔥 详细描述 RAG 完整流程（核心卖点）
- 🔥 强调混合检索 + Rerank（RAG 核心技术）
- 🔥 强调 Qdrant 技术细节（向量数据库能力）
